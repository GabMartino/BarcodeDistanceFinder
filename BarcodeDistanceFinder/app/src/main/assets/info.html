Information on Barcode Distance Finder

How does it work?

This application use the Mobile Vision library to detect barcodes from the image.
The barcodes detected have specific contour around them with some sizes. These information are used with a bit of arithmetic to capture distance information.





f(mm): Focal lenght of the camera.
Heighto(mm): Real height of the object that we want to calculate the distance with.
Heighti(pixels): Size of the image visualized. (This depends on the preview size selected from the device camera.
c Ratio between the resolution of the device and the resolution of the camera source.
Heighto(pixels): Object height in pixels, this are the values of the preview size.
Heights(pixels): Size of the sensor of the camera, specific for each device.

What does that information mean?

Quality If the device is correctly calibrated from the manufacturer to detect the distances '1' is shown, otherwise a lower value. In the latter case the distance calculated is unreliable.
MinFocusDistance This value represent the diopters of the minimum focus.(Diopter = 1/meters)
HyperFocalDistance This value represents the diopters of the maximum focus.
